# Hand Detection Project 🖐️

Proyek ini adalah aplikasi berbasis **Python** yang dirancang untuk mendeteksi tangan secara real-time. Proyek ini dikembangkan oleh **Po Weslay** sebagai bagian dari eksplorasi teknologi Computer Vision.

## 🚀 Fitur Utama
* **Real-time Detection:** Mendeteksi gerakan dan posisi tangan melalui kamera secara langsung.
* **High Accuracy:** Menggunakan library pendukung untuk memastikan deteksi yang responsif.
* **Easy Setup:** Struktur kode yang sederhana dan mudah dipahami untuk pengembangan lebih lanjut.

## 🛠️ Teknologi yang Digunakan
* **Python:** Bahasa pemrograman utama.
* **MediaPipe/OpenCV:** Library utama untuk pengolahan citra dan deteksi tangan.
* **PowerShell:** Untuk konfigurasi lingkungan kerja.
